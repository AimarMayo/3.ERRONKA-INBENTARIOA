Namespace Erronka_Interfazak

Classes

DBKonexioa

MySQL datu-basearekin konexioa kudeatzen duen klase estatikoa.

Erabiltzailea

Sistemako erabiltzaile baten eredua. Langileak eta administratzaileak biltzen ditu.

FAldatu

Gailua aldatzeko formularioa. ID bidez gailua bilatzen du eta bere datuak eguneratzeko aukera ematen du.

FEzabatu

Gailua ezabatzeko formularioa. ID bidez gailua bilatzen du, berrestea eskatzen du eta ezabatutakoen zerrendara mugitzen du.

FEzabatuakIkusi

Ezabatutako gailuak ikusteko formularioa. Historiko gisa gordetako ezabatutako gailu guztiak erakusten ditu.

FGehitu

Gailua gehitzeko formularioa. Ordenagailuak eta inprimagailuak datu-basean txertatzeko aukera ematen du.

FIkusi

Gailuak ikusteko formularioa. Inbentarioko gailu guztiak DataGridView batean erakusten ditu.

FInzidentziaEzabatu

Inzidentzia ezabatzeko formularioa. ID bidez inzidentzia bilatzen du, gailua ezabatutakoen zerrendara mugitzen du eta inzidentzia eta gailua datu-basetik ezabatzen ditu.

FInzidentziak

Inzidentziak ikusteko formularioa. Matxuratutako gailuen inzidentzia guztiak DataGridView batean erakusten ditu.

FKonpondu

Gailua matxuratuta markatzeko formularioa. ID bidez gailua bilatzen du, egoera 'matxuratuta' ezartzen du eta inzidentzia berri bat sortzen du.

FKonponduta

Gailua konponduta markatzeko formularioa. Inzidentzia ID bidez bilatzen du, gailuaren egoera 'erabilgarri' ezartzen du eta inzidentziaren konpontze-data gordetzen du.

FLangileaAldatu

Langilearen datuak aldatzeko formularioa. ID bidez langilea bilatzen du eta bere datuak eguneratzeko aukera ematen du.

FLangileaEzabatu

Langilea ezabatzeko formularioa. ID bidez langilea bilatzen du, berrestea eskatzen du eta datu-basetik ezabatzen du.

FLangileaGehitu

Langilea gehitzeko formularioa. Langile berri bat datu-basean txertatzeko aukera ematen du, emailaren eta mintegiburuaren bikoiztasuna egiaztatuz.

FLangileakIkusi

Langileak ikusteko formularioa. Sistemako langile guztiak DataGridView batean erakusten ditu.

FMenua

Aplikazioaren menu nagusia. Erabiltzailearen rolen arabera menuko aukerak gaitzen edo desgaitzen ditu.

FMintegiaEzabatu

Mintegia ezabatzeko formularioa. ID bidez mintegia bilatzen du eta berrestea eskatu ondoren datu-basetik ezabatzen du. Mintegiak lotutako langileak edo gailuak baditu, ezabatzea blokeatzen du.

FMintegiaGehitu

Mintegia gehitzeko formularioa. Mintegi berri bat datu-basean txertatzeko aukera ematen du, izen bikoizketak egiaztatuz.

FMintegiaIkusi

Mintegiak ikusteko formularioa. Sistemako mintegi guztiak DataGridView batean erakusten ditu.

FPasahitza

Pasahitza aldatzeko formularioa. Lehen saioan "123" pasahitz lehenetsia erabiltzen denean automatikoki irekitzen da. Gutxienez 8 karaktereko pasahitz berria eskatzen du.

FSarrera

Saioa hasteko formularioa. Erabiltzaileak emaila eta pasahitza sartuz sisteman sartzen dira.

Gailua

Inbentarioko gailu baten oinarrizko eredua. Ordenagailuak eta inprimagailuak klase honetatik heredatzen dira.

Galdera

Erabiltzaileari baieztapen-galdera egiteko elkarrizketa-koadro pertsonalizatua sortzeko klase estatikoa.

Mintegia

Lantegi edo mintegiaren eredua. Gailuak eta langileak mintegiarekin lotzen dira.

Saioa

Uneko erabiltzailearen saioa-datuak gordetzen dituen klase estatikoa. Saioa hasi ondoren, formulario guztiek klase honetatik irakurtzen dute erabiltzailearen informazioa.
